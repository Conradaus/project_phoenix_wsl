# Domain model template — DO NOT MODIFY

The canonical domain-model template lives at
[`.cursor/templates/domain-model.md`](../../.cursor/templates/domain-model.md).

When drafting a new domain doc:

1. Copy the template body.
2. Fill in every section; state diagrams are expected.
3. Write to `design/domain/<concept>.md`.
4. Run the Tier A gate before committing.
