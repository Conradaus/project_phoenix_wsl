# Prototype-spec template — DO NOT MODIFY

The canonical prototype-spec template lives at
[`.cursor/templates/prototype-spec.md`](../../.cursor/templates/prototype-spec.md).

Scope is hook-protected. Writes under `design/scope/**` trigger the
canonical-edit-gate hook and require explicit human approval. The
sanctioned path to create or edit a scope doc is the `derive-spec`
skill, which wraps its output in a `propose-adr` so scope changes
are logged as decisions.
